# 都市武俠：我的公司好像是邪教｜V4

Python + Streamlit + SQLite 手機優先互動小說。

- 20 名固定角色，每局 6 名核心 + 隨機 5 名支線
- 18 個主要決策，一局約 15～25 分鐘
- 60 門搞笑功法，從對話與事件選項中領悟
- 已學功法會依後續場景標籤變成特殊選項
- 功法使用後可能產生延遲後遺症
- 每局事件家族不重複
- 6 種深層秘密與多種結局方向
- 手機畫面只保留一個場景、3～4 個選項，人物／功法／線索放在 Popover

## 執行
```bash
pip install -r requirements.txt
streamlit run app.py
```

## GitHub 根目錄
`app.py`、`requirements.txt`、`game/`、`data/`、`tests/`、`scripts/` 都要在同一層。

## V4.4 逐行字幕對話版
- 選擇答案後，先播放玩家自己的選項台詞
- 當前事件的主要角色會產生 2 句角色專屬回應
- 字幕逐行累積出現，形成互動小說對話節奏
- 使用功法時，角色會再追加一段對該功法的專屬吐槽
- 字幕播放完後才接續結果與下一個事件
- 20 名角色皆有不同說話風格
- 對話資料庫共 5,760 筆
